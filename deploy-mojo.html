cics-bundle:deploy

Full name:

com.ibm.cics:cics-bundle-maven-plugin:0.0.2-SNAPSHOT:deploy

Description:

Deploys a CICS bundle to the specified CICS region using the CICS bundle deployment API. A matching bundle definition must be provided in the CSD in advance.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: verify.

Required Parameters

Name Type Since Description
<bunddef> String - The name of the bundle definition that will install this bundle. Must be present already in the CSD relating to the configured cicsplex/region, and must be configured with the correct bundle directory according to the bundle deployment API configuration and the name of the bundle the user is deploying.
<csdgroup> String - The CSD group containing the bundle definition to be installed.

Optional Parameters

Name Type Since Description
<bundle> String - The filename of the bundle archive file to be deployed.
<cicsplex> String - The name of the CICSplex the bundle should be installed into. Specifying this parameter overrides any value provided within a Maven settings server entry.
<classifier> String - The classifier of a bundle attached to this project which is to be deployed. If a value for the @bundle parameter is supplied, this classifier is ignored.
<password> String - The password to authenticate with. Specifying this parameter overrides any value provided within a Maven settings server entry.
<region> String - The name of the region the bundle should be installed into. Specifying this parameter overrides any value provided within a Maven settings server entry.
<serverId> String - The ID of a server configured in your Maven settings
<url> String - The full URL of the endpoint. Specifying this parameter overrides any value provided within a Maven settings server entry.
<username> String - The username to authenticate with. Specifying this parameter overrides any value provided within a Maven settings server entry.
